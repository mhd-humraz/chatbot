## 📌 Overview
The **Assistant Chatbot** is a simple and interactive web-based chatbot built using Python and Streamlit.  
It can hold basic conversations with users, respond to common queries, and provide helpful information through a clean chat interface.

---

## 🎯 Objective
To design and develop a chatbot capable of engaging in basic conversations using a user-friendly web interface.

---

## 🚀 Features
- 💬 Interactive chat interface  
- 🤖 Rule-based responses  
- 📚 Study tips and exam guidance  
- 😂 Fun jokes for engagement  
- ⚡ Fast and lightweight (no API required)  
- 🖥️ Clean UI using Streamlit  

---

## 🧠 Approach
This chatbot uses a **rule-based approach**, where predefined conditions and responses simulate conversation.

### ✅ Advantages:
- No cost (free to use)  
- Instant response time  
- Easy to build and modify
